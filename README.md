# Panel Tilt Visualizer

**Panel Tilt Visualizer** is a lightweight browser app for Nextracker solar-panel inspection robots: load robot telemetry CSVs, build odometry to visualize pitch along the path and per panel.
Developed by Blau Robotics. We build reliable tools for advanced control systems — because at Blau Robotics, **we do it best.**
